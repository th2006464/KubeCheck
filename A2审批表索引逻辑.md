# A2 审批表索引逻辑

本文档说明系统如何解析 Excel 格式的 A2 审批矩阵并建立搜索索引。当 A2 审批表更新时，按此文档的规则重新处理。

---

## 1. 输入文件

| 文件 | 说明 |
|------|------|
| `DSH-AGR-BPC-CP-001 ... Authority Matrix.xlsx` | 粮油事业部核决权限表 |
| `DSH-FOD-BPC-CP-001 ... 核决权限表A2.xlsx` | 食品事业部核决权限表 |

**放置位置**：`search\` 目录

---

## 2. Excel 结构

### 2.1 Sheet 组织

每个 Excel 包含多个 Sheet：
- **Sheet 1**：封面（`Approval sheet`）→ 跳过
- **Sheet 2**：说明页（`Instruction`）→ 跳过
- **Sheet 3+**：按业务分类的审批表，如：
  - `A2.1 General matters` — 计划及一般性事务
  - `A2.4 Procurement` — 采购
  - `A2.7 Seal Usage` — 用印
  - ...

### 2.2 每个 Sheet 的布局

```
Row 1-2: 标题
Row 3:   主表头（分组标签）
Row 4:   子表头
Row 5:   具体审批人角色名（如：部门负责人、CFO、CEO）
Row 6+:  数据行
         Col A (1):  序号（可为空）
         Col B (2):  编码 Code（如 A2.4.5）
         Col C (3):  分类（如"采购合同（采购金额小于1万...）"）
         Col D (4):  项目子项（如"直接采购"、"SAP合同创建和修改"）
         Col E (5):  金额条件（如 ≥0、<100）
         Col F (6):  申请人（如"采购部"）
         Col G+ (7+): 审批人列（单元格内 √ 表示该审批人参与）
```

### 2.3 关键特征

- **子行无编码**：子项行 Col B 为空，需继承上方最近一行的编码
- **子行无分类**：子项行 Col C 为空，需继承上方最近一行的分类
- **中英双语**：几乎所有单元格都包含中文 + 换行符 + 英文翻译
- **审批人标记**：审批人列用 `√` 或 `✓` 标记

---

## 3. 解析逻辑

### 3.1 列映射（固定，不猜）

| Excel 列 | 字段名 | 用途 |
|----------|--------|------|
| Col B (2) | Code | 编码，如 A2.4.5 |
| Col C (3) | 分类 | 业务分类名 |
| Col D (4) | 项目 | 具体子项描述 |
| Col E (5) | 金额 | 金额区间条件 |
| Col F (6) | 申请人 | 提交部门 |
| Col G+ (7+) | 审批人 | √ = 该角色参与审批 |

### 3.2 父级继承

按行号递增遍历，维护两个状态变量：

```
parentCode     ← 当 Col B 非空时更新
parentCategory ← 当 Col C 非空且非备注行时更新
```

子行（Col B/C 为空）继承最近的父级值。

### 3.3 审批人链构建

1. 从 Row 5 读取各列的审批人角色名
2. 对于每一行，遍历 Col G+（7 列及之后）
3. 单元格包含 `√` → 将该列对应的角色名加入审批链
4. 审批链格式：`角色1 → 角色2 → 角色3 → ...`

### 3.4 英文去除

所有文本字段在索引前调用 `KeepChinese()`：
- 按 `\n`（换行符）分割，取第一部分
- 通常中文在前，英文在后
- 适用对象：Code、分类、项目、金额、申请人、审批角色名、Sheet 名

### 3.5 无效行跳过

以下情况不建立索引：
- Col A 含 "序号" 或 "Code"
- Col B/C/D 全为空
- 审批人链为空（无 √ 标记的纯备注行）

### 3.6 最终显示格式

```
序号列：A2.4.5
名称列：A2.4.5 | 采购合同（采购金额小于1万...） | SAP合同创建和修改
金额/申请人：💰≥0 | 👤采购部
审批人列：部门负责人 → 法务 → CFO → CEO
```

## 4. 更新流程

当 A2 审批表新版发布时：

1. 将新 `.xlsx` 文件放入 `search\` 目录（覆盖旧文件）
2. 打开搜索页面 → 点击右侧「重新加载索引」
3. 索引重建完成，无需重启服务

**注意**：
- 文件名保持原有格式（DSH-AGR-...、DSH-FOD-...）
- Sheet 命名保持 `A2.1`、`A2.2` 等格式
- 如果新增 Sheet，系统自动识别并索引
- 如果新增审批人列（横向扩展），Row 5 新增的角色名会自动收录

## 5. 技术实现参考

核心代码在 `Models\SearchIndex.cs` 的 `LoadXlsx()` 方法中。

依赖库：[ClosedXML](https://github.com/ClosedXML/ClosedXML) 0.102.3（MIT 协议，无需安装 Excel）。

关键代码段：

```csharp
// 固定列映射
var code    = ws.Cell(r, 2).GetString().Trim();  // Col B
var category = ws.Cell(r, 3).GetString().Trim();  // Col C  
var desc    = ws.Cell(r, 4).GetString().Trim();  // Col D
var amount  = ws.Cell(r, 5).GetString().Trim();  // Col E

// 父级继承
if (!string.IsNullOrWhiteSpace(code)) parentCode = code;
if (!string.IsNullOrWhiteSpace(category) && !isNote) parentCategory = category;

// 审批人链
for (int c = 6; c <= lastCol; c++)
    if (ws.Cell(r, c).GetString().Contains("√"))
        chain.Add(approverHeaders[c - 6]);

// 英文去除
string KeepChinese(string s) {
    var idx = s.IndexOf('\n');
    return idx > 0 ? s[..idx].Trim() : s.Trim();
}
```
